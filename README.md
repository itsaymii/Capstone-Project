# 🚨 Disaster & Accident Incident Mapping System

## 📌 Overview
A web-based system designed to collect, manage, and visualize accident and disaster incident reports in real time.

## 🛠 Tech Stack
- React + TypeScript
- Tailwind CSS
- Django
- Django REST Framework
